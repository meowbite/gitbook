# quick start

- 使用docker快速启动一个Prometheus

***prom/proethues是官方镜像，建议使用的时候指定版本***

```bash
docker run -p 9090:9090 -v /prometheus-data \
       prom/prometheus:v2.13.1 --config.file=/prometheus-data/prometheus.yml
```

# 访问地址

```bash
http://localhost:9090
```
